# mdd-pyramid

ピラミッド図プラグイン

ピラミッド/階層図（マズローの欲求階層、戦略ピラミッドなど）をSVGとしてレンダリングします。

## 使い方

```
cat input.pyramid | mdd-pyramid > output.svg
```

## 入力形式

```
頂点レベル
第二レベル
第三レベル
基盤レベル
```

説明付き（ダイアグラムの右側に水平線で接続して表示されます）:

```
戦略 { 長期的な方向性 }
戦術 { 四半期の施策 }
実行 { 日々のオペレーション }
```

説明は複数行にも対応しています。`{` から `}` までが説明になります。

```
ビジョン {
  企業の存在意義
  長期的な目標
}
戦略 { 方向性の決定 }
実行 { 日々のオペレーション }
```
