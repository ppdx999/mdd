# mdd-table

テーブル（表）プラグイン

パイプ区切りの入力からSVGテーブルを生成します。

## 入力形式

```
| ヘッダー1 | ヘッダー2 | ヘッダー3 |
| セル1     | セル2     | セル3     |
| セル4     | セル5     | セル6     |
```

### 小計・合計行

`|=` で小計行、`|==` で合計行を挿入できます。
小計行はテーブルを分割し、テーブル枠の外側に独立して表示されます。

```
| 項目   | 金額      |
| 人件費 | 1,000,000 |
| 交通費 | 50,000    |
|= 小計  | 1,050,000 |
| 消耗品 | 200,000   |
| 通信費 | 100,000   |
|= 小計  | 300,000   |
|== 合計 | 1,350,000 |
```

### 列幅と折り返し

- 列の幅はテキスト量に比例して自動調整されます
- 最大幅（400px）を超えるテキストは自動的に折り返されます
- 行の高さは折り返し行数に応じて可変です

## 使い方

```sh
cat input.table | mdd-table > output.svg
```
