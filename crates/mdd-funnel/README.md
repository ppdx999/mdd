# mdd-funnel

ファネル図プラグイン。営業ファネルやコンバージョンファネルなど、段階的に絞り込まれるプロセスを視覚化します。

## 使い方

```
cat input.funnel | mdd-funnel > output.svg
```

## 入力形式

```
stage ラベル : 値
stage ラベル2 : 値2
```

値を省略すると、各段階が固定比率で順に狭くなります。

```
stage 認知
stage 興味
stage 検討
stage 購入
```

### 説明付き

各ステージに説明を追加できます。説明はダイアグラムの右側に水平線で接続して表示されます。

```
stage 認知 : "広告・SNSで知る"
stage 興味 : "サイト訪問・資料請求"
stage 検討 : "比較・評価"
stage 購入 : "契約・導入"
```

値と説明の両方を指定することもできます。

```
stage リード獲得 : 1000 : "広告・展示会"
stage 商談化 : 400 : "ヒアリング・提案"
stage 受注 : 40 : "契約締結"
```

説明は複数行にも対応しています。開き `"` から閉じ `"` までが説明になります。

```
stage 認知 : "広告・SNSで知る
サイト訪問
資料請求"
stage 検討 : "比較・評価"
```
