# mdd-mindmap

マインドマッププラグイン

中心トピックから放射状に広がるマインドマップ図をSVGとしてレンダリングします。

## 入力形式

```
center "中心トピック"
  ブランチ1
    サブアイテム1
    サブアイテム2
  ブランチ2
    サブアイテム3
```

- `center` 行で中心トピックを定義します。
- インデント（スペース2つ）で階層を表現します。レベル1がメインブランチ、レベル2がサブアイテムです。

## 使い方

```bash
cat input.mindmap | mdd-mindmap > output.svg
```
