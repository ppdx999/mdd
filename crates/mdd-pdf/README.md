# mdd-pdf

mdd で処理済みの Markdown を PDF に変換するツール。`# 見出し` でページを分割し、各ページのサイズはコンテンツに合わせて自動調整される。

## 使い方

```bash
# mdd で処理した Markdown を PDF に変換
mdd input.md | mdd-pdf > output.pdf
```

## ページ分割

`# 見出し` (h1) ごとにページが分割される。

```markdown
# 売上レポート

売上は前年比20%増。

```funnel
stage リード : 1000
stage 受注 : 40
```

# チーム構成

```org
member CEO
member CTO
CEO -> CTO
```
```

上記は 2 ページの PDF になる。各ページのサイズは含まれる図のサイズに合わせて自動調整される。

## 特徴

- `# 見出し` でページ分割
- 各ページのサイズはコンテンツに合わせて自動調整
- SVG ダイアグラムはセンタリングして配置
- テキストとダイアグラムを混在可能
- 高解像度レンダリング (2x)
