# mdd-list-v

縦型リスト図プラグイン。

番号付きバッジとラベル・説明文を縦に並べたリスト図をSVGとして生成します。番号なしリスト（箇条書き）にも対応しています。

## 入力形式

### 番号付きリスト

```
title "タイトル"
1. "ラベル" : "説明文"
2. "ラベル2"
```

### 番号なしリスト

```
title "タイトル"
- "ラベル" : "説明文"
- "ラベル2"
```

- `title` は省略可能です。
- `1.` や `-` で始まるアイテムが1つ以上必要です。
- `:` の後の説明文は省略可能です。
- 番号付きと番号なしは混在可能です。
- 番号付きリストの番号はDSL上の値に関わらず、出現順に自動採番されます。

## 使い方

```sh
cat input.list-v | mdd-list-v > output.svg
```
