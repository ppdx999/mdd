# 受注処理 DFD

```dfd
entity 顧客
entity 配送業者

process 受注処理
process 在庫引当
process 出荷指示

datastore 注文 {
  注文ID
  顧客ID
  注文日
  合計金額
  ステータス
}

datastore 商品 {
  商品ID
  商品名
  価格
  在庫数
}

datastore 出荷 {
  出荷ID
  注文ID
  追跡番号
  出荷日
}

顧客 -> 受注処理 : "注文リクエスト"
受注処理 -> 注文 : "注文登録"
受注処理 -> 在庫引当 : "在庫確認依頼"
在庫引当 -> 商品 : "在庫照会"
商品 -> 在庫引当 : "在庫情報"
在庫引当 -> 出荷指示 : "引当完了"
出荷指示 -> 出荷 : "出荷登録"
出荷指示 -> 配送業者 : "出荷依頼"
```
