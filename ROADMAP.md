# MDD プラグイン ロードマップ

資料作成で頻出する SmartArt 系の図をプラグインとして追加していく。

## Tier 1 — 高優先度

汎用性が高く、資料の「分析・判断・説明」パートで頻出する図。

- [ ] **mdd-matrix** — マトリクス図
  - 2軸で分類する図（重要度x緊急度、影響x確率など）
  - データ: 2軸のラベル + 各象限のアイテム
- [ ] **mdd-compare** — 比較図
  - 2〜3案の対比（現状 vs 提案、A案 vs B案）
  - データ: 列ごとのタイトル + 項目リスト
- [ ] **mdd-cycle** — サイクル図
  - 循環するプロセス（PDCA、開発サイクルなど）
  - データ: ノードの循環リスト
- [ ] **mdd-pyramid** — ピラミッド図
  - 階層構造の概念図（戦略→戦術→実行など）
  - データ: 上から下への層リスト
- [ ] **mdd-timeline** — タイムライン
  - 沿革、マイルストーン、ロードマップの時系列表示
  - データ: 日付 + イベントのリスト

## Tier 2 — 中優先度

特定の場面で重宝する図。

- [ ] **mdd-venn** — ベン図
  - 集合の重なり（スキルの共通部分、責任範囲など）
  - データ: 2〜3 の集合 + 各領域のラベル
- [ ] **mdd-funnel** — ファネル図
  - 営業パイプライン、コンバージョン漏斗
  - データ: 段階名 + 数値のリスト
- [ ] **mdd-radial** — 放射図（ハブ&スポーク）
  - 中心概念と周辺要素の関係
  - データ: 中心ノード + 周辺ノードリスト
- [ ] **mdd-process** — プロセス矢印
  - 3〜5ステップの業務手順を矢印で繋ぐ
  - データ: ステップ名 + 説明のリスト
- [ ] **mdd-kpi** — KPI / メトリクスカード
  - 数値ハイライト（売上、DAU、達成率など）
  - データ: ラベル + 値 + 変化率のリスト

## Tier 3 — あると便利

- [ ] **mdd-swot** — SWOT 図
  - 内部外部分析（マトリクスの特殊形）
  - データ: 4象限に項目リスト
- [ ] **mdd-mindmap** — マインドマップ
  - ブレスト、アイデア整理
  - データ: 中心トピック + 放射状の枝
