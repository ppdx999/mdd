# MDD プラグイン ロードマップ

資料作成で頻出する SmartArt 系の図をプラグインとして追加していく。

凡例: [x] = 実装済み、[ ] = 未実装

## フロー・プロセス系

- [x] **mdd-flowchart** — フローチャート（縦型）
- [ ] **mdd-process** — プロセス矢印（横型フロー）
  - 3〜5ステップの業務手順を横方向の矢印で繋ぐ
- [x] **mdd-swimlane** — スイムレーン図
- [ ] **mdd-cycle** — サイクル図
  - 循環するプロセス（PDCA、開発サイクルなど）
- [ ] **mdd-steps** — 階段・ステップ図
  - 段階的な進行・成長を階段状に表現する
- [ ] **mdd-funnel** — ファネル図
  - 営業パイプライン、コンバージョン漏斗
- [ ] **mdd-timeline** — タイムライン
  - 沿革、マイルストーン、ロードマップの時系列表示
- [x] **mdd-gantt** — ガントチャート

## 階層・構造系

- [x] **mdd-tree** — ツリー図
- [ ] **mdd-pyramid** — ピラミッド図
  - 階層構造の概念図（戦略→戦術→実行など）
- [ ] **mdd-layer** — 階層・レイヤー図
  - 積層構造（OSI参照モデル、アーキテクチャレイヤーなど）
- [ ] **mdd-triangle** — トライアングル図
  - 3要素の三角関係（QCD、スコープ・コスト・時間など）
- [x] **mdd-infra** — インフラ構成図

## 比較・分析系

- [ ] **mdd-matrix** — マトリクス図
  - 2軸で分類する図（重要度x緊急度、影響x確率など）
- [ ] **mdd-compare** — 項目比較図
  - 2〜3案の対比（現状 vs 提案、A案 vs B案）
- [ ] **mdd-scale** — 規模比較図
  - 数量や規模の大小を面積・長さで視覚的に比較する
- [ ] **mdd-analysis** — 規模分析図
  - 構成比や内訳を視覚化する（積み上げバー、ウォーターフォールなど）
- [x] **mdd-before-after** — ビフォーアフター図
  - 変更前後の状態を左右に並べて対比する
- [ ] **mdd-swot** — SWOT 図
  - 内部外部分析（マトリクスの特殊形）
- [ ] **mdd-ranking** — ランキング図
  - 順位付きリスト（売上ランキング、優先度順位など）

## 関係・概念系

- [ ] **mdd-venn** — ベン図
  - 集合の重なり（スキルの共通部分、責任範囲など）
- [ ] **mdd-radial** — 放射図（ハブ&スポーク）
  - 中心概念と周辺要素の関係
- [ ] **mdd-concept** — 相関図・概念図
  - 概念間の自由な関係性を線と矢印で表現する
- [ ] **mdd-mindmap** — マインドマップ
  - ブレスト、アイデア整理（中心トピック + 放射状の枝）
- [ ] **mdd-puzzle** — パズル・ハニカム構造図
  - ピースの組み合わせや蜂の巣状の要素配置で全体像を表現する
- [x] **mdd-er** — ER 図
- [x] **mdd-usecase** — ユースケース図
- [x] **mdd-dfd** — データフロー図
- [x] **mdd-state** — 状態遷移図
- [x] **mdd-sequence** — シーケンス図

## グループ・レイアウト系

- [ ] **mdd-group** — グループ図（少数要素）
  - 2〜4グループ程度の要素をまとめて配置する
- [ ] **mdd-group-multi** — グループ図（多数要素）
  - 多数のグループ・要素をグリッド状に整理して配置する
- [ ] **mdd-table** — 表
  - Markdown の表より視覚的にリッチな SVG テーブル

## リスト・箇条書き系

- [ ] **mdd-list-v** — 箇条書き（縦並び）
  - アイコン付きの縦方向リスト
- [ ] **mdd-list-h** — 箇条書き（横並び）
  - カード状の横方向リスト
- [ ] **mdd-list-grid** — 箇条書き（羅列）
  - グリッド配置の項目一覧
- [ ] **mdd-kpi** — KPI / メトリクスカード
  - 数値ハイライト（売上、DAU、達成率など）

## 特殊系

- [ ] **mdd-map** — 地図・マップ
  - 拠点配置や地理的関係を簡易的に表現する
- [ ] **mdd-math** — 数式
  - 数式をSVGでレンダリングする
